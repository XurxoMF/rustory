# [Welcome to Rustory](https://rustory.xyz)

Rustory is the unoficial launcher for Vintage Story, also known as VS Launcher 2.0, with almost all the features you need one click away! Managing Vintage Story Installations, Mods, Servers, Backups, etc... Check the next section for a complete features list!  
It works on Linux, Windows and MacOS.

## What do Rustory has to offer?

- Manage VS Installations
- Manage VS Servers
- Install, update and manage Mods
- Create, manage and share Modpacks
- Create, manage and restore Backups
- Multi-language
- Extremely optimiced with a very low use of RAM and CPU
- Multi-task
- Easy to install and use
