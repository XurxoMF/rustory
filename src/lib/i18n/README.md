# ADD A NEW LANGUAGE

To add a new language just follow this steps:

- Edit the `/rustory.inlang/settings.json` and add the locale on the `locales` array.
- Edit the `/src/lib/i18n/localesMeta.ts` and add the required info like `locale`, `name` and `credits`.

And that should be it. Just use Sherlock VS Code extension or [https://fink.inlang.com/](https://fink.inlang.com/) to translate it.
